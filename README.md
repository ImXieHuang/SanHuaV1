# SanHuaV1 大模型
**SanHuaV1 是一个全开源的个人实验性项目。** 该项目旨在挑战原有的自然语言处理架构，在保证模型正常运转的情况下，最小化所需的空间和算力，并且制造具有自主知识产权的全套方法链。
