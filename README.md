# SanHuaV1 仿生大模型
**SanHuaV1 是一个全开源的个人实验性项目。** 它并非传统意义上的大语言模型或多模态模型，而是一次对大脑皮层工作原理的解构与模拟尝试。

**该项目立足于联结主义和亚符号主义的学术思想，** 旨在通过构建类脑的神经网络结构（如结合反应-扩散模型与元胞自动机的思想），让机器能够以更接近生物的方式去“感知”和“思考”。
